# Native Bound Unbound CMS

Version 0.0.2

- Built with Sanity.io

## Running this repo

1. Clone
2. Uses PNPM, so get that and then run `pnpm i`
3. Run `santiy start`
    - This should spin up the local Sanity server and CMS
